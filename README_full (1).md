### 🤖 Agentic AI Projects

**[OceanShield AI — Real-Time Maritime Threat Detection](https://github.com/HARSHAN-7)**
`Python · Scikit-learn · Isolation Forest · Folium · National AI/ML Hackathon 2026`
Autonomous surveillance agent detecting illegal vessel activity (dark ships, loitering, AIS gaps) using Isolation Forest on live AIS data — achieved **<5% false-positive rate**. Weighted risk-scoring engine (Low/Medium/High) surfaced on a live Folium map, cross-referenced against simulated SAR satellite data. Presented architecture to 8 judges at a national AI/ML hackathon.

**Resume Analyzer AI — ATS Optimizer Agent**
`n8n · OpenAI API · Webhook · PDF Extraction`
Fully autonomous pipeline: webhook-triggered resume + job-description intake → PDF extraction → GPT-based ATS scoring and keyword-gap analysis → structured JSON output, with conditional branching and error handling for edge cases — zero manual intervention.

**Social Media AI Content Generator Agent**
`n8n · OpenAI API · Instagram Graph API`
End-to-end agentic workflow: brand input via webhook → AI-generated post copy → media container creation → autonomous publishing to Instagram → verification loop, with dynamic branching for post-published vs. content-only outcomes.

### 📊 Data Analytics Projects

**Customer Churn Prediction & Retention Analytics** · Kaggle, 7,043 rows · [GitHub](https://github.com/HARSHAN-7/Customer-Churn-Prediction-Retention-Strategy-Using-Machine-Learning)
Random Forest model (**87% accuracy vs. 73% baseline**) flagging ~15% high-risk customers into plain-language risk tiers; feature-importance analysis on top churn drivers (tenure, contract type, usage); full Power BI retention dashboard with drill-through filters.

**AI-Driven Healthcare Diagnosis & Prediction** · UCI ML datasets · [GitHub](https://github.com/HARSHAN-7/AI-in-Healthcare-Diagnosis-and-Prediction-using-Machine-Learning)
**88–92% classification accuracy** across diabetes, heart disease, and breast cancer prediction tasks. Compared 4 model types, selecting on recall priority to minimize false negatives — the clinically costlier error.

**Retail Sales Forecasting & Demand Analytics (DMart)** · 180K records · [GitHub](https://github.com/HARSHAN-7/Dmart-Analysis---Data-Science-Project)
Regression forecasting models achieving an **18% RMSE reduction**, translated into reorder thresholds, seasonal buffer-stock timing, and outlet-level pricing sensitivity bands.

**HR & Business Performance Analytics** · [GitHub](https://github.com/HARSHAN-7/HARSHAN-End-to-End-Business-Performance-Analytics-HR-Analytics-)
End-to-end workforce analytics covering attrition, hiring efficiency, and compensation trends for data-driven HR decisions.
